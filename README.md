## Endpoints

| Method | Endpoint             | Description                                       |
| ------ | -------------------- | ------------------------------------------------- |
| GET    | `/chapter?id=1`      | Returns a chapter with the given id.              |
| GET    | `/chapters`          | Returns all chapters.                             |
| GET    | `/bookChapters?id=1` | Returns all chapters of a book with the given id. |
